# Common Images

Place shared images that are used across all locales here.

## Purpose:
This folder contains images that are **language-neutral** and used universally across all versions of the site (en_IE, uk_UA, pt_BR).

## Image Guidelines for Common Assets:
- **Language-neutral**: No text in images
- **Culturally neutral**: Avoid region-specific symbols or themes
- **Universal icons**: Logo, generic icons, system graphics
- **Shared UI elements**: Buttons, dividers, patterns

## Suggested Common Images:
- `logo.png` - CrossPaths logo
- `logo-white.png` - White version for dark backgrounds  
- `favicon.ico` - Browser favicon
- `default-avatar.png` - Generic user avatar
- `icon-*.png` - UI icons (search, filter, notifications, etc.)
- `pattern-*.png` - Background patterns or textures

## When to Use Common vs. Locale-Specific:
- **Use common/** for: logos, icons, UI elements, brand assets
- **Use locale folders** for: hero images, cultural photos, localized graphics
