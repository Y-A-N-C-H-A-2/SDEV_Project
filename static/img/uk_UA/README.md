## Image Guidelines for uk_UA locale:
- **Style**: Formal, structured, trustworthy
- **Tone**: Professional, reassuring, detailed
- **Colors**: Muted blues, greys, whites - calming and authoritative
- **Content**:
  - Verified host badges and credentials
  - Safe, well-lit public venues
  - Structured event settings (arranged seating, clear organization)
  - Official-looking materials
  - Small, intimate group settings
  - Clear signage and wayfinding
